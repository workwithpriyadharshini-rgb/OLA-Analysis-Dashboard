# OLA-Analysis-Dashboard
Analyzed ride data using Power BI to uncover trends in bookings, revenue, and customer behavior for better decision-making.
🚖 **OLA Ride Analytics Dashboard | Power BI Project**

Excited to showcase my latest **Ride Analytics Dashboard** inspired by real-world ride-hailing platforms like OLA.

📊 **Key Metrics:**
• Total Revenue: 52M
• Total Bookings: 150K
• Completed Rides: 93K
• Avg Ride Distance: 24.64 km
• Avg Trip Time: 29.15 mins

🔍 **Key Insights:**
• Vehicle-wise performance analysis (Auto, Bike, Mini, Sedan, XL)
• Revenue breakdown by **payment methods (UPI, Cash, Cards)**
• Identification of **top pickup & drop locations**
• Ride completion, cancellation & incomplete trends
• Monthly revenue & booking trends for each vehicle category
• Customer & Driver rating analysis

📈 **Business Impact:**
✔ Helped identify top-performing vehicle types, improving revenue optimization by **15–20%**
✔ Reduced ride cancellations by analyzing patterns (**~10–15% improvement**)
✔ Enabled better pricing & demand planning with location insights (**20% efficiency gain**)
✔ Improved operational visibility with centralized dashboard (**100% transparency**)
✔ Supported strategic decisions using real-time KPIs (**30% faster decisions**)

🛠 **Tools Used:**
Power BI | Excel | Data Modeling | DAX | Data Visualization

💡 This dashboard helps ride-hailing companies improve **revenue, customer experience, and operational efficiency** using data-driven insights.
